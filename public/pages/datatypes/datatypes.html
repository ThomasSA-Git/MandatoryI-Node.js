Data types

In JavaScript the type of variable is defined by the value it holds unlike other programming languages that has them explicitly declared (String, int, boolean and so on). Instead they are declared with const, let and var.

  • const (immutable): const is used for variables whose values should not change after initialization. It is used for values that represent constants or unchanging data.
    Example: const pi = 3.14;
  • let (Mutable): let is used for variables whose values can be changed. It is the preferred choice when a variables value may change during while running the program.
    Example: let counter = 0;
  • var: var was used in older versions of JavaScript to declare variables. It has functional scope (scoped to the nearest function), global scoping when set out of a function and some hoisting behavior. It is less commonly used now with const and let being preferred due to their block scope and more predictable behavior.

As a rule you should never use var due to its unpredictable behaviour. Instead you should use const when possible and let when it is necessary to change the value of a variable during execution of the program.


Primitive Data Types

1. String: Represents text characters and is enclosed in single (' ') / (` `) or double (" ") quotes. 

            
                const message = 'Hello, World!';
                const message = `Hello, World!`;
                const message = "Hello, World!";

There are several methods for handling strings in JavaScript. Below is a list of some:

  • length(): Returns the length of the string.
  • charAt(): Returns the character at the specified index.
  • concat(): Combines two or more strings and returns a new string.
  • indexOf(): Returns the index of the first occurrence of a substring in the string.
  • substring(): Extracts a portion of the string between the specified start and end indexes.
  • toUpperCase(): Converts the string to uppercase.
  • toLowerCase(): Converts the string to lowercase.
  • trim(): Removes whitespace from the beginning and end of the string.
  • replace(): Replaces occurrences of a substring with another substring.
  • split(): Splits the string into an array of substrings based on a specified separator like ",", " " and so on.


2. Number: Represents numeric values, fx int or float numbers. 

            
                const num = 42;

As with strings, numbers has several methods too. Below is a list of some:

  • toFixed(): Formats a number to a specified number of decimal places.
  • toString(): Converts a number to a string.
  • parseInt(): Parses a string and returns an integer.
  • parseFloat(): Parses a string and returns a floating-point number.
  • isNaN(): Checks if a value is NaN (Not-a-Number).
  • Math.round(): Rounds a number to the nearest integer.
  • Math.floor(): Rounds a number down to the nearest integer.
  • Math.ceil(): Rounds a number up to the nearest integer.

3. Boolean: Represents true or false values.

            
                const isTrue = true;
                const isFalse = false;

4. BigInt: Represents large integers.

            
                const bigIntValue = 1234567890123456789012345678901234567890n;

Special Data Types

5. Undefined: Represents a variable that has been declared but hasn't been assigned a value. 

            
                let undefinedVar;

6. null: Represents the intentional absence of any object or value. 

            
                const emptyValue = null;

Symbol

7. Symbol: Represents a unique and immutable value that is often used as an object property key. 

            
                const uniqueSymbol = Symbol('description');


Objects, Arrays, and Dates in JavaScript

1. Objects: Objects in JavaScript are collections of key-value pairs where each key is a string and each value can be of any data type. 

        
            const person = {
                name: 'Thomas',
                age: 39,
                isStudent: true
            };

2. Arrays: Arrays are ordered lists of values (including objects) in JavaScript. Elements in an array can be of different data types. 

        
            const colors = ['red', 'green', 'blue'];

3. Dates: Dates in JavaScript represent date and time information. They are useful for working with timestamps, clocks, countdowns and managing calendar projects. 

        
        const currentDate = new Date();
